# laravel-vuejs
This is a simple project made in laravel and nuxt js.

To run this project give the following commands:

- Laravel:

composer install
php artisan key:generate
php artisan migrate:fresh --seed
php artisan passport:install
php artisan serve

- Vue Js:

cd client
npm install
npm run dev
